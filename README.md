# WealthPass 💼

WealthPass is a smart financial management web platform designed to simplify claim handling, record management, and digital access through a clean and user-friendly dashboard.

## 🚀 Features

- 🔐 Secure user authentication
- 📄 Claims management system
- 📊 Smart dashboard
- 📁 File and document handling
- 🎨 Clean responsive interface
- ⚡ Fast and scalable backend

## 🛠 Tech Stack

- Python
- Django
- HTML
- CSS
- JavaScript
- SQLite3

## ▶️ How to Run Locally

1. Clone the repository

git clone https://github.com/asif-build/WealthPass.git

2. Enter project folder

cd WealthPass

3. Create virtual environment

python -m venv .venv

4. Activate virtual environment

Windows:

.venv\Scripts\activate

5. Install dependencies

pip install -r requirements.txt

6. Run development server

python manage.py runserver
